**Helper Tool – Feature Overview**

A lightweight developer utility for inspecting project files, visualizing folder structures, and securely storing development secrets.

---

### Core Modes

**Code Mode**
Fetches and displays the contents of all files within a selected directory.

**Folder Mode**
Retrieves and displays the full folder structure of a selected project.

---

### Display Modes

**Roof Mode (Basic View)**
File explorer style interface similar to Visual Studio Code.

* Expandable folders
* Simple hierarchical view

**Tree Mode (Structured View)**
Displays folders and files as a tree structure.

* Folders act as nodes
* Files appear as leaf elements
* Provides a structural overview of the project

---

### Functional Features

**File Extension Detection**

* **Detect Mode** – Automatically detects and processes file types based on extension.
* **Ignore Mode** – Skips selected file extensions.

**Folder Detection**

* **Focus Mode** – Processes only selected folders.
* **Ignore Mode** – Excludes specified folders from processing.

---

### Security

**Secret Holder Mode**

Secure storage for sensitive development information.

* Password required before access
* Allows storage of API keys, JWT tokens, credentials, or other secrets depending on user needs.
