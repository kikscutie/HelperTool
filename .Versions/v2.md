# Overview

Version 2 enhances search accuracy and introduces file filtering controls for more precise and flexible file handling.

This version focuses on usability improvements, better filtering, and improved workflow efficiency.

---

# Improvements

* Improved readability of search results
* More accurate file matching
* Supports directory-based searching (e.g., `Backend/src/models`) instead of matching only file names

---

# New Features

* File extension selection to detect only chosen file types

* Useful for targeting specific extensions (e.g., `.js`, `.ts`, `.cs`)

* Reduces unnecessary results

* Ignore extension selection to temporarily exclude chosen file types

* Provides more flexible filtering when scanning folders

* Refresh function to reload the current folder structure

* Useful when files are added, removed, or modified

* Eliminates the need to restart the application
