Version 2

Start Date: March 4, 2026
Release Date: March 4, 2026

Overview

Version 2 enhances search accuracy and introduces file filtering controls for more precise and flexible file handling.

This version focuses on usability improvements, better filtering, and improved workflow efficiency.

Improvements
1. Enhanced File Searching

Improved readability of search results.

More accurate matching.

Supports directory-based searching (e.g., Backend/src/models) instead of matching only file names.

New Features
1. File Extension Selection

Detects only selected file extensions.

Useful for targeting specific file types (e.g., .js, .ts, .cs).

Reduces unnecessary results.

2. Ignore Extension Selection

Temporarily excludes selected file extensions from scanning.

Allows more flexible filtering when analyzing folders.

3. Refresh Function

Reloads the current folder structure.

Useful when files are added, removed, or modified.

Eliminates the need to restart the application.

Notes

Version 2 improves control, precision, and efficiency, making the tool more practical for structured and large-scale projects.
