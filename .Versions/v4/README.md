# Version 4

**Start Date:** March 7, 2026
**End Date:** MArch 7, 2026

# Overview

Version 4 focuses on **customization, productivity tools, and improved visual experience**.

This update introduces a **Settings Mode** where users can configure application preferences such as themes. It also adds a new **Notepad Mode**, allowing users to quickly write and store notes directly within the application.

Additionally, the overall **design and visual experience** have been improved to provide a cleaner and more modern interface.

---

# New Features

### Settings Mode

A new **Settings Mode** has been added, allowing users to customize application preferences.

Current settings include:

* Theme selection
* Visual customization options

This provides users with greater control over how the application looks and behaves.

---

### Theme Selection

Users can now select from multiple visual themes.

Available themes include:

**Dark Themes**

* Navy Dark
* Catppuccin Mocha
* Solarized Dark
* GitHub Dark

**Light Themes**

* Cream Light
* Catppuccin Latte
* Solarized Light
* GitHub Light

Users can quickly toggle between **light and dark theme counterparts**, allowing smooth switching depending on environment or preference.

This feature improves visual comfort during long development sessions.

---

### Notepad Mode

A new **Notepad Mode** has been introduced to allow users to write and manage notes directly inside the application.

Possible uses include:

* Development reminders
* Temporary notes
* Task tracking
* Code snippets
* Ideas or documentation

This feature helps users **capture thoughts quickly without leaving the tool**.
