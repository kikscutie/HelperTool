Version 1

Start Date: January 5, 2026
Release Date: January 8, 2026

Overview

Version 1 is the first stable release of Helper Tool.
This version introduced the core functionality for project structure analysis and code extraction.

Features
1. Fetch Code from Selected Files

Extracts and displays code from selected files.

Allows faster review without manually opening each file.

2. File Searching

Search files by name inside the selected directory.

Helps quickly locate specific files in large projects.

3. Folder Structure Fetching

Generates and displays the full folder structure.

Useful for understanding project architecture at a glance.

Notes

This release focuses on foundational functionality and usability.
It establishes the base system for future improvements.
